# CIFAR-10 Image Classification (ResNet-18 Transfer Learning)

## 🚀 Results
- Test Accuracy: 93.30%
- Test Loss: 0.2018

## 🔧 Techniques
- Transfer Learning (ResNet-18)
- Data Augmentation (RandomCrop, HorizontalFlip)
- Adam Optimizer
- CrossEntropy Loss

## 📊 Dataset
CIFAR-10 (10 classes)

## 🧠 Key Insight
Transfer learning significantly improves performance over CNN models trained from scratch.
